# Royal Bites - Restaurant Website

Royal Bites is a fully responsive and modern restaurant website built using HTML, CSS, and JavaScript. The project is designed to provide users with a complete restaurant browsing experience including menu exploration, food ordering, table reservation, contact functionality, shopping cart system, checkout process, and responsive navigation for all devices.

The website features a professional UI/UX inspired by premium restaurant websites with smooth animations, interactive sections, responsive layouts, and mobile-friendly navigation.
